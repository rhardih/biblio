Simple plugin to showcase your personal reading interesests, library etc.

== Description ==

Biblio adds a simple interface to the admin panels for keeping track of the stuff you read. You can indicate current readings as well as keep an archive of what you've already finished reading.

As well as keeping track of your readings, biblio also offers the possibility to save any notes you feel like jotting down for future keeping. 

Besides the admin panels, biblio can also add a small sidebar widget, to list your currently active readings.
